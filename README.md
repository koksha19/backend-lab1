# Backend labs

## Description
This repo contains labs for Backend development
## Install & Run Locally
```bash
npm install
npm run start:dev
```

## Run with Docker
```bash
docker build -t backend-labs .
docker run -p 3000:3000 backend-labs
```

## Run with Docker Compose
```bash
docker-compose up --build
```
